# Домашнее задание skillfactory - модуль 7
---
Контейнер данных _IntegerArray_ — объект, который хранит целые числа и способен вычислить собственный размер во время выполнения программы.
Ваш код будет оцениваться ментором курса, а значит, вы сможете получить вместе с баллом развёрнутый комментарий эксперта. Оцениваться задание будет максимум в  баллов **по трём критериям**: 

1. Безопасность кода: все случаи исключений поддержаны (создание или копирование неправильного размера контейнера, выход за границы контейнера), код безопасен для использования;
2. Функциональность контейнера: мы предлагаем вам создать контейнер целых чисел наподобие [IntArray](https://www.learncpp.com/cpp-tutorial/106-container-classes/). Помимо безопасности исключений, контейнер также должен обладать функциональностью с возможностями:
    
    - создать контейнер;
    - скопировать контейнер;
    - получить доступ к любому элементу контейнера по индексу;
    - изменить размер контейнера;
    - вставить элемент в контейнер;
    - удалить элемент из контейнера.
3. Дополнительный балл можно получить за реализацию вставки в начало и конец и поиска в контейнере элемента по значению.
4. Оформление кода (объявление и реализация классов).
